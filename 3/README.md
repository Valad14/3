# Histori Corpus — версия без автоматических типов

Что изменено:

1. Убран фильтр по типам.
2. Убраны автоматические колонки `Тип (...)` и `Причина (...)`, потому что классификация морфологических, графических, фонетических и других разночтений работала ненадёжно.
3. Убрана видимая колонка `Разметка эталона` из таблицы.
4. Название `эталонный список` заменено на `основной список`: он нужен только как база для выравнивания строк.
5. В XML-экспорте больше не записывается атрибут `variant` с автоматическим типом разночтения.
6. Оставлены выравнивание слов, контекст, статистика найденных соответствий и режим совместимости символов для старых компьютеров.
7. Сохранено исправленное чтение XML-файлов с неверно указанной кодировкой.

Запуск:

```bash
pip install -r requirements.txt
streamlit run app.py
```

Для обновления репозитория замените файл `3/app.py` на этот `app.py`.
