# Histori Corpus — исправленная версия

Что изменено:

1. Исправлено чтение XML, где в декларации стоит `encoding="UTF-16"`, но файл фактически сохранен в UTF-8.
2. Переписана классификация типов разночтений: отдельно проверяются графика, фонетика, морфология, лексика и пропуски.
3. При чтении XML больше не теряются повторные значения `f/symbol`, например несколько `case` или `gender`.
4. Добавлен фильтр по типам: можно показывать выбранные типы в любом списке или только в выбранном списке.
5. Добавлен режим совместимости символов для старых компьютеров: оригинал, без надстрочных знаков, совместимый режим.
6. В таблице типы можно исправлять через выпадающий список.
7. Исправлен контекст для неэталонных списков: теперь подсвечивается слово именно выбранного источника.

Запуск:

```bash
pip install -r requirements.txt
streamlit run app.py
```

Для обновления репозитория замените файл `3/app.py` на этот `app.py`, а `requirements.txt` оставьте как в архиве.
