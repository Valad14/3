# Histori Corpus — яркий Streamlit-интерфейс

В этой версии изменен только внешний вид приложения: добавлена яркая цветовая тема, стилизация боковой панели, кнопок, таблицы, блока контекста, метрик и экспандера с инструкцией.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Что осталось прежним

- загрузка XML-файлов;
- выбор эталонного списка;
- парсинг XML-TEI;
- алгоритм выравнивания;
- классификация разночтений;
- редактирование таблицы;
- просмотр контекста;
- экспорт CSV и XML-TEI ZIP.

## Примечание о шрифтах

В архив не включены бинарные или закодированные файлы шрифтов. Интерфейс переведен на системный стек шрифтов с fallback для исторической кириллицы.
